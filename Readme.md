### Project Description
RolledQR is a professional QR code generator that allows users to create QR codes for various types of content, including URLs, text, contact information, and WiFi credentials, all in a user-friendly interface but the twist is all qr gen are for rick roll video.

### The Problem (that doesn't exist)
In a world where QR codes are everywhere, why not make it easier for people to generate them without the hassle of complicated software?

### The Solution (that nobody asked for)
RolledQR simplifies the QR code creation process by providing a straightforward web application that generates QR codes in multiple formats with just a few clicks. It's fun, fast, and free!
